# Tumor Detection Project

## How to Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/TheBeerkat/Tumor-Detection.git
   cd Tumor-Detection
   ```

2. **Run the setup script:**
   ```bash
   ./setup.sh
   ```

3. **Run the application:**
   ```bash
   python src/main.py
   ```
